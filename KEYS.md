Use these keys to control the application or navigate categories.

| Key | Action |
| :--- | :--- |
| **`R`** | **Refresh feeds** (Checks all RSS links for new videos) |
| **`S`** | **Toggle Shorts** (Hide/Show videos shorter than 60s) |
| **`+`** | **Add Channel** (Paste an RSS/Channel URL) |
| **`-`** | **Delete Channel** (Remove a feed from your list) |
| **`M`** | **Mark All Seen** (Marks all current videos as read) |
| **`L`** | **Watch Later** (Open your saved playlist) |
| **`Q`** | **Quit** the application |
| **`?`** | **Help** (Show this guide) |

> **Tip:** You can type to filter! E.g., type "linux" to find a channel.

### âš¡ Action Menu (After selecting a video)
Once you press `Enter` on a video, you can choose:

1.  **Play (QuickTube)** - Copies link & launches QuickTube player.
2.  **Add to Watch Later** - Saves video to your local playlist.
3.  **Open in Browser** - Opens link in your default web browser.
4.  **Remove** - (Only inside Playlists) Removes the video from the list.
